# 快捷工具 - 快速开始

## 立即开始使用

### 1. 基础命令（无需学习）

- `/clear` - 清空聊天
- `/总结` - 聊天总结
- `/导出` - 导出记录

### 2. 群聊使用

在群聊中，需要先@机器人：

```
@机器人 /clear
@机器人 /总结
```

### 3. 私聊使用

直接发送命令即可：

```
/clear
/总结
```

## 查看所有命令

发送 `/状态` 获取完整命令列表

## 常用操作流程

### 开始新对话
1. `/新会话` - 创建新会话
2. `/切换 gemma3:12b` - 选择本地模型
3. 开始正常对话

### 整理重要内容
1. `/总结` - 总结今天对话
2. `/导出` - 保存到本地
3. `/当前` - 查看统计

### 学习使用所有功能
1. `/状态` - 查看所有命令
2. `/历史 10` - 浏览历史
3. `/当前` - 了解统计

## 常见问题

**Q: 为什么命令不响应？**
A: 群聊需要@机器人，私聊直接发送即可

**Q: 导出文件在哪里？**
A: `session/chat_exports/` 目录

**Q: 如何切换模型？**
A: 发送 `/切换` 查看可用，然后 `/切换 模型名`

**Q: 清空会话会丢失什么？**
A: 会清空当前所有聊天记录（除非已导出）
