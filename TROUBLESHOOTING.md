# 命令使用问题排查指南

## 常见问题

### 1. 命令没有反应

**可能原因**:
- 在群聊中没有@机器人
- 命令输入格式错误
- 机器人未正常运行

**解决方法**:
```
❌ 错误: 直接在群聊发送 /总结
✅ 正确: @机器人 /总结
✅ 私聊: 直接发送 /总结
```

### 3. 机器人不响应

**检查步骤**:
1. 确认机器人正在运行
2. 查看机器人日志输出
3. 检查命令是否正确输入

## 调试步骤

### 1. 查看机器人日志

```bash
# 启动机器人时会看到日志输出
python bot.py
```

### 2. 检查命令处理器

确保看到以下调试信息:
```
DEBUG: 消息处理器收到消息，用户: xxx, 消息: /总结
DEBUG: /总结命令被触发，用户: xxx
```

### 3. 验证命令注册

运行诊断脚本:
```bash
python diagnose_commands.py
```

## 测试命令

### 基础测试

1. 私聊发送 `/状态`
   - 应该返回系统状态和命令列表

2. 私聊发送 `/总结`
   - 应该返回聊天总结

3. 私聊发送 `/清空`
   - 应该返回"已清空聊天记录"

### 群聊测试

1. 在群聊中@机器人 `/状态`
2. 在群聊中@机器人 `/总结`
